
# Behavioral Analytics Dashboard

This project demonstrates a Power BI dashboard analyzing behavioral data through KPI tracking and user segmentation.

## Overview
The dashboard focuses on understanding how different user segments vary in decision-making behavior.

## Key Features
- KPI: Average Decision Behavior Score
- Segmentation: High vs Low Maladaptive groups
- Comparison: Behavioral differences across segments
- User distribution by segment

## Tools Used
- Power BI (data visualization)
- SQL (aggregation and segmentation queries)
- Excel/CSV (data source)

## Data Workflow
Data was processed through a simplified analytics pipeline:
CSV → transformation → aggregation → visualization

## Key Insight
Users in the high maladaptive segment show different decision behavior patterns compared to low maladaptive users, highlighting the importance of segmentation in behavioral analysis.
